# 🤔 What is Time and Work?

Time and Work is based on a very simple idea:

> **More Efficiency → Less Time**
> **Less Efficiency → More Time**

For example:

* If Rahul finishes a job in **10 days**, he completes **1/10** of the work in one day.
* If Priya finishes the same job in **5 days**, she completes **1/5** of the work in one day.

This daily work done is called **Efficiency** or **Work Rate**.

---

# 📚 Basic Concepts

## 1️⃣ Work

The total task to be completed.

Example:

* Painting a wall
* Building a house
* Completing a software project

---

## 2️⃣ Time

The number of days, hours, or minutes required to complete the work.

---

## 3️⃣ Efficiency

Efficiency means **how much work a person completes in one unit of time**.

### Formula

```text
Efficiency = Work ÷ Time
```

---

# 🧮 Important Formulas

## Formula 1

```text
Work = Time × Efficiency
```

---

## Formula 2

```text
Efficiency = Work ÷ Time
```

---

## Formula 3

```text
Time = Work ÷ Efficiency
```

---

## Formula 4

If a person completes a job in **N days**

Then,

```text
One Day's Work = 1/N
```

Example

If A finishes a job in 8 days

Then

```text
A's One Day Work = 1/8
```

---

# 📊 Understanding Combined Work

Suppose

A completes a job in **12 days**

B completes the same job in **6 days**

Then

A's One Day Work

= 1/12

B's One Day Work

= 1/6

Together

```text
1/12 + 1/6

= 3/12

= 1/4
```

Therefore,

Together they finish the work in

**4 days**

---

# 💡 Memory Tricks

🧠 **Golden Rule**

> Faster worker ⇒ Higher efficiency ⇒ Less time.

---

🧠 **One-Day Work Trick**

If a person takes

10 days

Always write

```text
1/10
```

Never start multiplying directly.

---

🧠 **Team Work Trick**

Whenever two people work together,

Always **ADD** their one-day work.

---

# ⚡ Shortcut Tricks

### Shortcut 1

If

A = 20 days

B = 30 days

LCM

= 60 Units

A

= 3 Units/day

B

= 2 Units/day

Together

= 5 Units/day

Time

= 60 ÷ 5

= **12 days**

This LCM method is much faster than working with fractions.

# 📝 Solved Example

### Example 1

Rahul completes a project in **15 days**.

How much work does he complete in one day?

### Solution

One Day Work

```text
= 1/15
```

✅ Answer

**1/15**

---

### Example 2

A completes a work in **10 days**

B completes it in **15 days**

Find the number of days required if they work together.

Solution

One Day Work

```text
1/10 + 1/15

= 3/30 + 2/30

= 5/30

= 1/6
```

Therefore,

Total Time

= **6 days**

---

# 💼 Placement Tips

* Always calculate **One Day Work** first.
* Use the **LCM Method** whenever possible.
* Practice Combined Work questions daily.
* Learn efficiency-based shortcuts to save time.

---

# ❌ Common Mistakes

* Mixing up Time and Efficiency.
* Forgetting to add one-day work for combined workers.
* Using different units (days vs hours) without conversion.
* Ignoring the LCM shortcut.
