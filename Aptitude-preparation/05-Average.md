# 🤔 What is Average?

An **Average** is a single value that represents a group of numbers.

It is also known as the **Arithmetic Mean**.

For example,

Suppose five students scored:

```text
70, 80, 90, 60, 100
```

Instead of remembering every score individually, we can represent them with one value called the **Average**.

---

# 🧮 Average Formula

**Average = Sum of all observations ÷ Number of observations**

---

# 📝 Example 1

Find the average of:

```text
10, 20, 30, 40, 50
```

### Solution

Sum

= 10 + 20 + 30 + 40 + 50

= 150

Number of observations

= 5

Average

= 150 ÷ 5

= **30**

✅ Answer = **30**

---

# 📝 Example 2

Find the average of:

```text
15, 18, 20, 22, 25
```

Sum

= 100

Average

= 100 ÷ 5

= **20**

# 📌 Important Formulas

### 1. Average

Average = Sum ÷ Number

---

### 2. Sum

Sum = Average × Number of Items

---

### 3. Number of Items

Number = Sum ÷ Average

---

# 💡 Shortcut Tricks

### Trick 1

If all numbers are equal,

their average is the same number.

Example

```text
20,20,20,20

Average = 20
```

---

### Trick 2

If one number increases by **x**,

the average increases by

```text
x ÷ Total Numbers
```

Example

There are 5 students.

One student's marks increase by 10.

Average increases by

10 ÷ 5

= **2**

---

### Trick 3

If every number increases by the same amount,

the average also increases by the same amount.

Example

```text
10,20,30

Average =20

Now add 5 to every number

15,25,35

Average =25
```

Average increased by **5**.

# 💼 Placement Tricks

* Memorize the three basic formulas.
* Never calculate unnecessarily.
* Use multiplication instead of repeated addition whenever possible.
* Practice age and salary average questions regularly.

---

# ❌ Common Mistakes

* Forgetting the total number of observations.
* Adding numbers incorrectly.
* Confusing total with average.
* Using the wrong formula.
