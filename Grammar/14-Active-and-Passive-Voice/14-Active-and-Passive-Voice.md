# 📖 What is Voice?

**Voice** shows whether the **subject performs the action** or **receives the action**.

There are **two types of Voice**:

* ✅ Active Voice
* ✅ Passive Voice

---

# 🟢 What is Active Voice?

In **Active Voice**, the **subject performs the action**.

### Structure

```text
Subject + Verb + Object
```

### Examples

✅ Rahul writes a letter.

✅ She teaches English.

✅ They completed the project.

---

# 🔵 What is Passive Voice?

In **Passive Voice**, the **subject receives the action**.

### Structure

```text
Object + Helping Verb + V3 + by + Subject
```

### Examples

✅ A letter is written by Rahul.

✅ English is taught by her.

✅ The project was completed by them.

---

# 🎯 Why Learn Active & Passive Voice?

Active and Passive Voice is important for:

* ✔️ Placement Exams
* ✔️ SSC, Banking & UPSC
* ✔️ Spoken English
* ✔️ Professional Emails
* ✔️ Report Writing
* ✔️ Interviews

---

# ⚖️ Active Voice vs Passive Voice

| Active Voice                | Passive Voice               |
| --------------------------- | --------------------------- |
| Subject performs the action | Subject receives the action |
| More direct                 | More formal                 |
| Common in conversations     | Common in reports and news  |
| Subject + Verb + Object     | Object + Helping Verb + V3  |

# 🔄 Steps to Convert Active to Passive

## Step 1

Find the **Subject**.

Example:

Rahul writes a letter.

Subject → Rahul

---

## Step 2

Find the **Object**.

Object → Letter

---

## Step 3

Move the object to the beginning.

A letter...

---

## Step 4

Use the correct **Helping Verb**.

Example:

Is / Am / Are / Was / Were / Has Been / Have Been / Will Be

---

## Step 5

Use the **Past Participle (V3)** form of the verb.

Write → Written

Clean → Cleaned

Eat → Eaten

Make → Made

---

## Step 6

Add **by + Subject** (only when necessary).

Final Sentence:

A letter is written by Rahul.

# 📚 Tense-wise Conversion Rules

## 1️⃣ Simple Present Tense

### Active Structure

```text
Subject + V1 + Object
```

### Passive Structure

```text
Object + is/am/are + V3 + by + Subject
```

### Examples

✅ Active: She writes a story.

✅ Passive: A story is written by her.

---

## 2️⃣ Present Continuous Tense

### Active

```text
Subject + is/am/are + V-ing + Object
```

### Passive

```text
Object + is/am/are + being + V3
```

### Examples

✅ Active: They are building a bridge.

✅ Passive: A bridge is being built by them.

---

## 3️⃣ Present Perfect Tense

### Active

```text
Subject + has/have + V3 + Object
```

### Passive

```text
Object + has/have + been + V3
```

### Examples

✅ Active: She has completed the task.

✅ Passive: The task has been completed by her.

---

# 📝 Important Note

Only **transitive verbs** (verbs that take an object) can be converted into passive voice.

### Convertible

✅ She wrote a letter.

(Object = Letter)

---

### Not Convertible

❌ He sleeps.

(No object)

❌ Birds fly.

(No object)

---

# ⚠️ Common Mistakes

### Mistake 1

❌ A letter is write by Rahul.

✅ A letter is written by Rahul.

---

### Mistake 2

❌ The work is completed from him.

✅ The work is completed by him.

---

### Mistake 3

❌ The homework has completed.

✅ The homework has been completed.
