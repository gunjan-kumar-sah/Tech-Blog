# 🔵 Synonyms

## 📖 What are Synonyms?

**Synonyms** are words that have the **same or nearly the same meaning**.

---

## Common Synonyms

| Word | Synonym |
|------|----------|
| Happy | Joyful |
| Big | Large |
| Small | Tiny |
| Fast | Quick |
| Smart | Intelligent |
| Begin | Start |
| End | Finish |
| Easy | Simple |
| Difficult | Hard |
| Beautiful | Attractive |
| Brave | Courageous |
| Honest | Truthful |
| Rich | Wealthy |
| Help | Assist |
| Buy | Purchase |

---

## Example Sentences

✅ She is a **smart** student.

✅ She is an **intelligent** student.

# 🔴 Antonyms

## 📖 What are Antonyms?

**Antonyms** are words with **opposite meanings**.

---

## Common Antonyms

| Word | Antonym |

| Rich | Poor |
| Success | Failure |
| Strong | Weak |
| Early | Late |
| Increase | Decrease |
| Accept | Reject |
| Maximum | Minimum |
| Victory | Defeat |
| Ancient | Modern |
| Honest | Dishonest |
| Expand | Contract |
| Arrival | Departure |
| Optimistic | Pessimistic |
| Possible | Impossible |

# 🟢 Collocations

## 📖 What are Collocations?

A **collocation** is a natural combination of words that native English speakers commonly use together.

---

## Common Collocations

| Correct | Incorrect |
|----------|------------|
| Make a decision | Do a decision |
| Take a break | Make a break |
| Have breakfast | Eat breakfast *(possible, but less common in formal collocation lists)* |
| Pay attention | Give attention |
| Catch a bus | Take a bus *(both are acceptable depending on context)* |
| Keep a promise | Maintain a promise |
| Save time | Store time |
| Heavy rain | Strong rain |
| Strong coffee | Powerful coffee |
| Deep sleep | Strong sleep |

---

## Example Sentences

✅ Please **pay attention** to the instructions.

✅ We need to **make a decision** today.

✅ She had a **deep sleep** after the exam.

---

# 🟣 Homophones

## 📖 What are Homophones?

**Homophones** are words that **sound the same but have different meanings and spellings**.

---

## Common Homophones

| Word 1 | Word 2 |
|---------|---------|
| Two | Too |
| To | Two |
| Right | Write |
| Sea | See |
| Son | Sun |
| Flower | Flour |
| Buy | By |
| Hear | Here |
| Meet | Meat |
| Pair | Pear |

---

## Examples

✅ I want to **buy** a laptop.

✅ Please stand **by** me.

# 🟡 Frequently Confused Words

| Word | Meaning |
|------|---------|
| Accept | Receive |
| Except | Excluding |
| Affect | Influence (Verb) |
| Effect | Result (Noun) |
| Advice | Suggestion (Noun) |
| Advise | Suggest (Verb) |
| Lose | Fail to keep |
| Loose | Not tight |
| Principal | Head of a school / Main |
| Principle | Rule or belief |
| Complement | Completes something |
| Compliment | Praise |

# 💼 Placement Vocabulary Tips

Many aptitude tests ask:

- Synonyms
- Antonyms
- One-word substitutions
- Word meanings
- Fill in the blanks
- Context-based vocabulary

### Preparation Tips

✔ Learn words in context.

✔ Read editorials and quality articles.

✔ Practice daily quizzes.

✔ Revise difficult words weekly.
