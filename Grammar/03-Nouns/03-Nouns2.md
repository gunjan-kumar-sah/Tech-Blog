# 3️⃣ Collective Noun

## 📖 Definition

A **Collective Noun** is a word that represents a **group of people, animals, or things as a single unit**.

In simple words:

> **A collective noun is the name of a collection or group.**

---

# 🌍 Examples of Collective Nouns

| Group             | Collective Noun |
| ----------------- | --------------- |
| Group of students | Class           |
| Group of players  | Team            |
| Group of soldiers | Army            |
| Group of animals  | Herd            |
| Group of birds    | Flock           |
| Group of flowers  | Bouquet         |
| Group of ships    | Fleet           |

---

# 📝 Sentences

### Example 1

> The **team** won the match.

Team → Collective Noun

---

### Example 2

> A **flock** of birds is flying in the sky.

Flock → Collective Noun

---

### Example 3

> The **army** protected the country.

Army → Collective Noun

---

# 💡 Memory Trick

Remember:

```text id="collective01"
Many People/Things Together = Collective Noun
```

Examples:

```
Players → Team

Students → Class

Soldiers → Army
```

---

# 4️⃣ Abstract Noun

## 📖 Definition

An **Abstract Noun** represents an idea, feeling, quality, state, or emotion that cannot be physically touched or seen.

It exists only in our thoughts or feelings.

---

# 🌍 Examples of Abstract Nouns

| Abstract Noun | Meaning |
| ------------- | ------- |
| Love          | Feeling |
| Honesty       | Quality |
| Happiness     | Emotion |
| Freedom       | State   |
| Knowledge     | Idea    |
| Wisdom        | Quality |
| Beauty        | Quality |

---

# 📝 Sentences

### Example 1

> **Honesty** is the best policy.

Honesty → Abstract Noun

---

### Example 2

> Everyone wants **happiness**.

Happiness → Abstract Noun

---

### Example 3

> **Knowledge** is power.

Knowledge → Abstract Noun

---

# 🧠 Memory Trick

Ask yourself:

> "Can I touch it?"

If the answer is **No**, it may be an Abstract Noun.

Examples:

```
Book → Can touch ❌ Abstract

Love → Cannot touch ✅ Abstract
```

# 6️⃣ Countable Noun

## 📖 Definition

A **Countable Noun** is a noun that can be counted.

It has:

* Singular form
* Plural form

---

# 🌍 Examples

| Singular | Plural    |
| -------- | --------- |
| Book     | Books     |
| Student  | Students  |
| Car      | Cars      |
| Apple    | Apples    |
| Computer | Computers |

---

# 📝 Sentences

✅ I have three books.

Books can be counted.

---

✅ She bought two laptops.

Laptops can be counted.

---

# 7️⃣ Uncountable Noun

## 📖 Definition

An **Uncountable Noun** is a noun that cannot be counted individually.

Usually, they represent:

* Liquids
* Materials
* Abstract ideas

---

# 🌍 Examples

| Uncountable Noun |
| ---------------- |
| Water            |
| Milk             |
| Sugar            |
| Rice             |
| Information      |
| Knowledge        |
| Advice           |

---

# 📝 Sentences

✅ I need some water.

❌ I need two waters.

---

✅ She gave me some advice.

❌ She gave me an advice.

---

# ⚖️ Countable vs Uncountable Noun

| Countable           | Uncountable       |
| ------------------- | ----------------- |
| Can be counted      | Cannot be counted |
| Has singular/plural | Usually no plural |
| Uses many/few       | Uses much/little  |

---

# Examples:

### Countable:

> I have many books.

---

### Uncountable:

> I don't have much time.

---

# 💡 Identification Tricks

## Trick 1

If you can put a number before it:

```
One book
Two books
Three books
```

It is Countable.

---

## Trick 2

If you need measurement:

```
A glass of water

A kilo of rice

A piece of advice
```

It is usually Uncountable.

# ⚠️ Common Mistakes

### Mistake 1

❌ Informations

✅ Information

Reason:

Information is uncountable.

---

### Mistake 2

❌ Advices

✅ Advice

Reason:

Advice has no plural form.

---

### Mistake 3

❌ Two breads

✅ Two pieces of bread

Reason:

Bread is uncountable.
