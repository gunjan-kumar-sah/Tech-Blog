# Versioning Policy

This project uses **Semantic Versioning**.

## Version Format

```text id="0c6z9h"
MAJOR.MINOR.PATCH
```

### MAJOR

Breaking changes or significant restructuring.

Example:

```text id="r7myc1"
2.0.0
```

### MINOR

New topics, new folders, or additional features.

Example:

```text id="dfv1mx"
1.1.0
```

### PATCH

Bug fixes, typo corrections, formatting improvements, and documentation updates.

Example:

```text id="v8p7nq"
1.0.1
```
